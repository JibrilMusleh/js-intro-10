

Functions: it is a block of code that can be executed repeatedly and increase the code reusability.
-it always runs a task
-There are two types of functions
    void: does not return the value and just runs the task 
    return: return a value (data) when invoked
-Functions can be static
    static: can be invoked with its class name or prototype
    Math.min()
    non-static: can be invoked witha n instance of the template
